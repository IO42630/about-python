# Collections

## Pandas

### Axis

- `axis=0` is the vertical axis, rows
  - concat,  means adding new rows 
- `axis=1` is the horizontal axis, columns
  - concat, means adding new columns
- `axis=2` is the depth axis, 3D
  - concat, means adding new depth

### Masking Operators
- `&` : and
- `|` : or
- `~` : not
